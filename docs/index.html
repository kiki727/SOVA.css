Helpers


Margins and Paddings

There is margin and padding class helpers. Pattern is:
  • p- : for all paddings
  • pl- : for padding-left
  • pr- : for padding-right
  • pt- : for padding-top
  • pb- : for padding-bottom
  • px- : for padding-left and padding-right
  • py- : for padding-bottom and padding-top
Values are from 0 to 5, where 0 is no padding ( margin ) ( padding: 0; )

.

Same applies for margin, just instead of p , you use m ( m-0, mt-4, etc...)
